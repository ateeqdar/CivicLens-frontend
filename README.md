# CivicLens Frontend

This repository contains the frontend application for CivicLens, built with React and Vite.

## Project Setup

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```

3.  **Build for Production:**
    ```bash
    npm run build
    ```

## Deployment

This frontend application is designed to be deployed as static assets. Common deployment platforms include Netlify, Vercel, GitHub Pages, etc.
